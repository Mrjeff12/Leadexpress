## 1. Product Definition

### One-liner
"AI that reads your WhatsApp groups and sends you only the jobs that fit you."

### Core Value Proposition
Contractors are in 10-30+ WhatsApp groups but can't read them all. Lead Express scans every message, identifies real job requests, matches by trade + county, and delivers them straight to WhatsApp — so contractors can work instead of scroll.

### Two Value Streams

**Stream 1 — Group Scanning (passive)**
1. Customer shares WhatsApp group invite links
2. Lead Express joins manually (within 24 hours)
3. AI scans every message, identifies job requests
4. Matches leads by trade + county
5. Sends to customer via WhatsApp: job description + poster details + "Contact them" button
6. Customer clicks → WhatsApp chat directly with the poster

**Stream 2 — Job Forwarding via Rebeca (active)**
1. Contractor sends Rebeca: "I have a job to forward"
2. Rebeca collects: trade, county, urgency, job description
3. Rebeca sends to ALL matching contractors (trade + county)
4. Contractor clicks → WhatsApp chat directly with the person who posted

### Key Principle
Lead Express is an information broker. We never touch the transaction. We connect people and get out of the way.

## 2. Pricing Model — "Base + Unlocks"

### Day 1: Two States Only

| | **FREE** | **PREMIUM $79/mo** |
|---|---|---|
| Add WhatsApp groups | Unlimited | Unlimited |
| AI scans your groups | ✅ | ✅ |
| Lead notifications | Teasers only (max 3/week) | **Full details + contact info** |
| Forward jobs via Rebeca | ✅ | ✅ |
| Receive forwarded jobs | Teasers only | **Full details** |
| Give lead feedback | ✅ | ✅ |
| Earn Network Points | x1 | **x2** |
| Dashboard | — | **Full + weekly report** |
| Support | Community | **Priority** |
| Free trial | — | **7 days** |

### Teaser Message (non-paying users)
When a matching lead is found for a non-paying user:
> "There's an HVAC job in Miami-Dade that matches your profile. Posted 12 minutes ago. Upgrade to see the details and contact the poster."

Max 3 teaser messages per week — enough to create FOMO, not enough to feel like spam.

### Groups Lifecycle
- Customer adds a group → it stays in the system **forever** (even if customer stops paying)
- Groups are always scanned
- Paying customer → gets full lead details
- Non-paying customer → gets teasers + "upgrade to see"
- This means the pool only grows, never shrinks

### Why No Tiers
- Contractors on a roof don't want to compare 3 plans
- "Yes or No" converts better than "which one?"
- One price = no confusion, no buyer's remorse
- Upgrade path comes through add-ons and Network Levels (future), not plan complexity

## 3. Network Points System

### How Points Are Earned

| Action | Points | Purpose |
|---|---|---|
| Add a WhatsApp group | +10 | Grows the network |
| 30 consecutive days active | +50 | Retention |
| Refer a friend (signs up) | +100 | Acquisition |
| Give lead feedback | +5 | Data quality |
| Report confirmed scammer | +25 | Network safety |
| Forward a job via Rebeca | +10 | Creates leads for others |

Premium users earn at **x2 rate**.

### Levels and Perks

| Level | Points Required | ~Time to Reach (Premium) | Day 1 Perk |
|---|---|---|---|
| **Member** | 0 | Immediate | Base experience |
| **Insider** | 200 | ~2 months | Lead quality score (hot/warm/cold) on every lead |
| **Partner** | 500 | ~4 months | Weekly group insights ("Group X is most active, most jobs in county Y") |
| **VIP** | 1,000 | ~8 months | **Unlocked in Phase 2** — Full network access |

### Design Principles
- Levels exist from Day 1 — customer sees progression immediately
- VIP is visible but perk is hidden → curiosity without false promises
- Perks unlock with engagement, not money → rewards loyalty
- When new features launch, loyal users already qualify

## 6. Phased Growth Plan

### Phase 1: "The First 500" (Month 1-6)

**Model:** Free + Premium $79/mo
**Focus:** Product-market fit, viral loop through job forwarding

**Growth engine:**
```
Paying user adds groups → gets leads → forwards jobs he can't take
→ Rebeca sends to matching contractors (including non-users)
→ Non-user gets: "Someone forwarded you a job. Reply YES for details."
→ Gets the job for free → "Want more? Start your 7-day trial."
→ CAC = $0
```

**Month-by-month targets:**

| Month | Paying | Free | Groups in Pool | MRR |
|---|---|---|---|---|
| 1 | 20 | 40 | 150 | $1,580 |
| 2 | 30 | 100 | 300 | $2,370 |
| 3 | 70 | 250 | 600 | $5,530 |
| 4 | 130 | 500 | 1,200 | $10,270 |
| 5 | 250 | 900 | 2,000 | $19,750 |
| 6 | **500** | **1,500** | **3,000+** | **$39,500** |

**Key metrics to track:**
- Paying users count
- Free users count
- Total groups in pool
- MRR
- Leads delivered / week
- Jobs forwarded / week
- Free → Paid conversion rate (target: 25%)
- Teaser → Upgrade conversion rate
- Churn rate (target: < 8%/month)
- Average points per user
- Groups added per user
- Feedback given per lead

**What to build:**
- [x] Landing page with correct messaging
- [x] Onboarding: trade + county + group links + WhatsApp connect
- [x] Lead scanning + WhatsApp notifications
- [x] Forward jobs via Rebeca
- [x] Teaser messages for free/expired users (max 3/week)
- [x] Network Points (earning + levels display)
- [x] Feedback system (got job / not relevant / scam)
- [x] Stripe billing: Free + Premium $79 + 7-day trial
- [x] Admin Growth Dashboard (see section 7)

### Phase 2: "The Network" (Month 6-12)

**Model:** Free + Premium $79 + Network Pass $39/mo add-on
**Focus:** Monetize the pool, reward loyal users

**What changes:**
- VIP level unlocks: "Full Network Access" — leads from ALL pool groups, not just yours
- New add-on: Network Pass ($39/mo) for users who aren't VIP yet
- Loyal customers (VIP) get it for free — rewarded, not punished
- New customers pay $79 + $39 = $118, or earn their way to VIP

**Targets:**

| Month | Paying | Avg Revenue/User | MRR |
|---|---|---|---|
| 6 | 500 | $79 | $39,500 |
| 9 | 800 | $95 | $76,000 |
| 12 | **1,500** | **$100** | **$150,000** |

**What to build:**
- [ ] Network pool matching (leads from other users' groups)
- [ ] Network Pass add-on (Stripe)
- [ ] VIP auto-unlock logic
- [ ] "Leads from network" vs "Leads from your groups" distinction in UI

---

### Phase 3: "The Marketplace" (Year 2)

**Model:** Lower subscription ($49) + transaction fees (5-8%)
**Focus:** Become the platform where jobs get done

**What changes:**
- Job bidding: poster gets 3-5 quotes from matching contractors
- Transaction fee: 5-8% on deals closed through Lead Express
- Verified Contractor badge: based on feedback score
- Premium subscription drops to $49 (more accessible, volume play)
- Revenue shifts from subscriptions to transactions

**Targets:**

| Metric | Target |
|---|---|
| Paying users | 5,000 |
| Deals/month | 2,000 |
| Avg transaction fee | $50 |
| Subscription MRR | $245,000 |
| Transaction MRR | $100,000 |
| **Total MRR** | **$345,000** |
| **ARR** | **~$4.1M** |

## 8. Future-Proofing Principles

1. **Never remove value from existing customers** — only add new value at new prices
2. **Loyal users get new features through Network Points** — they earned it
3. **New users can pay to skip** — or earn their way up
4. **Subscription is the floor, not the ceiling** — transaction fees are the real revenue engine
5. **Every user (free or paid) makes the network more valuable** — never discourage free users
6. **Data is the long-term moat** — lead quality, contractor reliability, market pricing intelligence
